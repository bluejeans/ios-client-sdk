VideoState

public enum VideoState : Equatable

Represents the current state of connection between the client application’s video media, and the connection to a BlueJeans meeting.

  • active

    Video is in active state

    Declaration

    Swift

    case active
  • inactive(_:)

    Video is not active, has an associated VideoStateInactiveReason value.

    Declaration

    Swift

    case inactive(VideoStateInactiveReason)
